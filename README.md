# Analizador de Temperaturas Mensuales

## Descripción
Este proyecto consiste en una aplicación desarrollada en Java que permite analizar temperaturas registradas durante un mes.

## Contexto
Proyecto académico realizado como parte de mis estudios de Desarrollo de Aplicaciones Web (DAW), con el objetivo de practicar programación estructurada sin programación orientada a objetos.

## Tecnologías
- Java
- Arrays
- Estructuras de control (if, switch, bucles)
- GitHub

## Funcionalidades
- Introducción de temperaturas manual o aleatoria
- Cálculo de la media
- Identificación de la temperatura máxima y mínima
- Generación de histograma
- Listado de días por encima de la media

## Documentación
El repositorio incluye un documento PDF con la explicación completa del proyecto, estructura y demostración del funcionamiento.

## Cómo ejecutar
1. Abrir el proyecto en un IDE como IntelliJ, VS Code
2. Ejecutar el archivo `Main.java`
3. Seguir las opciones del menú en consola
